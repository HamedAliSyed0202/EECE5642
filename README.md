### Multi-Metric Difficulty Visualization & Cluster Analysis
**EECE 5642 Data Visualization · Northeastern University · Prof. Y. Raymond Fu**

---

## 📋 Project Overview

This project presents a systematic **visual profiling framework** for the KITTI odometry benchmark. Standard SLAM benchmarking practice reports results as flat numerical tables — making it impossible to answer multi-dimensional questions like *"which sequence is hardest, and why?"* or *"are any metrics redundant?"*

We extract **7 heterogeneous metrics** from all 11 KITTI ground-truth pose sequences, apply K-Means clustering (k=3), and reveal findings through **5 complementary visualizations** that are completely invisible in tabular data.

**Core argument:** 4 of 6 key analytical questions about SLAM benchmarking are unanswerable from raw tables. Visualization is the primary research methodology.

```bash
pip install numpy pandas matplotlib seaborn scikit-learn scipy
```

### Step 2 — Configure the dataset path
Open `Analysis/kitti_analysis.py` and set line ~54:
```python

```
The script expects:
```
<KITTI_BASE_PATH>/
  poses/
    00.txt  01.txt  02.txt  ...  10.txt
```

### Step 3 — Run the analysis
```bash
cd Analysis
python kitti_analysis.py
```

**Output:** All 7 figures + `kitti_metrics_summary.csv` saved to `Outputs/`

### Step 4 — Open the dashboard
1. Open `kitti_dashboard_v5.html` in **Chrome** (or any modern browser)
2. Click **📁 Browse File** on the Option A card
3. Select `Outputs/kitti_metrics_summary.csv`
4. Click the green **▶ Compute Dashboard** button
5. All charts update with your real data

## 📐 The 7 Metrics

| # | Metric | Formula | What it captures |
|---|--------|---------|-----------------|
| 1 | **Step Variance** | σ(‖tᵢ₊₁ − tᵢ‖) | Motion irregularity — frame-to-frame distance std dev |
| 2 | **Drift Slope** | Linear regression of d(t) | Rate of drift growth over the sequence |
| 3 | **Accumulated Drift** | ‖t_N − t₀‖ | Final displacement from start (Seq 01: 1,980 m) |
| 4 | **Early Error** | mean(dᵢ), first half | Average displacement in first half |
| 5 | **Late Error** | mean(dᵢ), second half | Average displacement in second half |
| 6 | **Path Curvature** | mean\|κᵢ\| of XZ path | Geometric complexity (Seq 07: 23.49, Seq 01: 0.005) |
| 7 | **Motion Entropy** | H = −Σ pₖ log₂ pₖ | Heading diversity (18-bin histogram) |

All metrics are **Min-Max normalised** to [0, 1] for unit-independent comparison.  
Composite score: `S = mean(7 normalised metrics)`

## 🖥️ Interactive Dashboard Features

The dashboard (`kitti_dashboard_v5.html`) provides:

| Panel | Description |
|-------|-------------|
| **Upload** | Load CSV from Python or raw pose .txt files directly |
| **Key Findings** | 4 auto-computed insight cards (difficulty ratio, r value, cluster split, PCA %) |
| **Weight Explorer** | 7 live sliders to reweight metrics + 3 presets (Drift-Heavy, Geometry-Heavy, Temporal-Heavy) |
| **Difficulty Ranking** | Horizontal bar chart, updates live with weight changes |
| **Radar Chart** | All 7 metrics overlaid per cluster |
| **Parallel Coordinates** | D3-powered with axis brushing/filtering |
| **Correlation Heatmap** | Full 7×7 Pearson matrix, max-r cell highlighted |
| **PCA Scatter** | Browser-computed PC1 vs PC2, click points to highlight |
| **Temporal Error** | Line chart showing 3 growth patterns (monotonic / oscillating / V-shape) |
| **Trajectory Viewer** | Animated ground-truth driving paths per sequence |
| **Sequence Profiles** | Card grid — one card per sequence with mini metric bars and insight text |
| **Metrics Table** | Sortable table of all raw values |

**Cross-chart linking:** Click any bar, point, or sequence card to highlight that sequence across all charts simultaneously.

## 🏗️ Clustering Details

**Algorithm:** K-Means, k = 3, n_init = 20 (multi-restart), convergence-checked  
**Feature matrix:** 11 sequences × 7 normalised metrics  
**Silhouette coefficient:** ≈ 0.61 (optimal over k = 2, 3, 4, 5)

| Cluster | Sequences | Score Range | Characteristics |
|---------|-----------|-------------|-----------------|
| **Easy** | 03, 04, 06 | 0.137 – 0.222 | Low drift, simple geometry, short routes |
| **Medium** | 00, 02, 05, 07, 08, 09, 10 | 0.258 – 0.444 | Urban routes, moderate complexity |
| **Hard** | 01 only | 0.793 | Straight highway, 1,980 m drift, zero curvature |
